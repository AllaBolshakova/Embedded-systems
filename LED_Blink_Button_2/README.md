Мигание диодом реализовано через опрос в while(1). Если кнопка нажата (is_pressed), но не была нажата ранее (!was_pressed), 
то для следующего такта считаем, что кнопка была нажата ранее (was_pressed = 1), но недолго (long_press_executed = 0) и начинаем таймер press_start_time = HAL_GetTick(). 
Если кнопка была нажата ранее и таймер превысил нужное значение (!long_press_executed && (HAL_GetTick() - press_start_time >= 1000)), то меняем режим скважности. 
Если кнопка не нажата, но была нажата и недолго, то меняем частоту мигания freq_idx++. 

Прерывание по таймеру void HAL_TIM_PeriodElapsedCallback(TIM_HandleTypeDef *htim) инвертирует состояние светодиода с нужной сважностью. 
Для скважности есть два режима: режим MODE_50_PERCENT, когда диод горит половину периода и MODE_50_MS, когда диод горит 50 миллисекунд. 

Для запуска кода нужно собрать проект, например, сочетанием ctrl+b, а затем прошить файл Embedded-systems\LED_Blink_Button\Debug\LED_Blink_Button_2.elf на плату, я прошивала через CubeProgrammer.
